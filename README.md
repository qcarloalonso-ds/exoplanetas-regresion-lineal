   # Predicción de Masa de Exoplanetas con Regresión Lineal

   Modelo de Machine Learning que predice la masa de exoplanetas usando características orbitales y estelares. Basado en metodología NASA Exoplanet Archive.

   ## Objetivo
   Demostrar pipeline completo de ML: análisis exploratorio, entrenamiento, validación y visualización aplicado a datos astronómicos.

   ## Stack Técnico
   `Python` `Pandas` `Scikit-learn` `Matplotlib` `Seaborn` `Regresión Lineal` `Validación Cruzada`

   ## Resultados Clave
   - **R² Score**: >0.85 en datos de prueba
   - **Métricas**: RMSE y MAE para evaluar precisión
   - **Visualización**: Gráfico de valores reales vs predichos

   ## Cómo ejecutarlo
   ```bash
   pip install -r requirements.txt
   python exoplanetas_regresion_lineal.py
```
